# 대량메일발송

기술지원본부 주관 메일발송 시스템

### versions
- JDK: 21
- Kotlin Gradle Plugin: 2.1.0
- Spring Boot: 3.5.4
- PostgreSQL

### 산출물
- Swagger ()
- OAuth2 인증 플로우
- 개발가이드

### 기타
환경 변수/민감 정보는 별도 파일(application-prod.yml) 에서 관리 (DB, Token 등)

### API문서 자동화 
- Spring RestDocs + Swagger
- https://mail.rsup.at/swagger-ui/index.html 로 진입 후 확인 가능
